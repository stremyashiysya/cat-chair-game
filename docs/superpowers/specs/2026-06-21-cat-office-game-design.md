# Дизайн: "Котик в офисе" — интерактивный офисный хаос

_Дата: 2026-06-21_

## Концепция

Sandbox-игра на основе Telegram-стикера. Игрок кликает предметы в офисе — кот подходит, зажимаешь кнопку и дубасит предмет, отпускаешь — предмет улетает с физикой. Офис накапливает хаос. Цели нет — чистое удовольствие от разрушения.

---

## Технический стек

- **Pixi.js v7** (CDN) — WebGL рендер, 60fps
- **Чистый JS** — никаких фреймворков
- **Физика** — собственная: гравитация + дуга + отскок (коэффициент 0.6)
- **Спрайты** — 16 кадров PNG, извлечённых из стикера (400px)
- **Один HTML-файл** — всё встроено, деплой на GitHub Pages

---

## Архитектура

```
index.html
├── Pixi.Application (canvas)
│   ├── Layer: background (статичный офис из кадра стикера)
│   ├── Layer: objects (интерактивные предметы)
│   ├── Layer: cat (спрайт кота, движется по сцене)
│   ├── Layer: flying (предметы в полёте, с физикой)
│   └── Layer: ui (счётчик, кнопка reset)
└── GameState
    ├── cat: { pos, state, target, chargeTime }
    ├── objects: Map<id, ObjectState>
    └── physics: тик гравитации каждый кадр
```

## Кот — состояния и спрайты

| Состояние | Кадры стикера | Описание |
|---|---|---|
| `idle` | 1–3 | Сидит за столом |
| `walk` | 4–7 | Пружинящий чиби-шаг к цели |
| `beat` | 8–12 | Удары пока держат |
| `throw` | 12–14 | Бросок при отпускании |
| `aftermath` | 13–15 | Довольный хаос |
| `return` | 16 | Возврат к столу |

**Движение**: кот идёт по прямой к цели, скорость 180px/s, bounce offset ±4px по Y (чиби-физика).

**Приоритет**: если кликнуть новый объект пока кот идёт — он меняет цель мгновенно. Если кликнуть пока кот бьёт — удар прекращается, объект остаётся на месте (не летит), кот идёт к новому.

## Интерактивные объекты (10 штук)

| ID | Объект | Позиция | Масса | Время заряда |
|---|---|---|---|---|
| `chair` | Офисное кресло | Рядом со столом | 5.0 | 2.0s до max |
| `monitor` | Монитор | На столе | 3.0 | 1.5s |
| `keyboard` | Клавиатура | На столе | 1.5 | 0.8s |
| `mouse` | Мышка | На столе | 0.5 | 0.3s |
| `papers` | Папки с бумагами | На столе | 1.0 | 0.5s, разлетаются листами |
| `pc` | Системный блок | Под столом | 8.0 | 3.0s |
| `cabinet` | Шкаф | У стены | 12.0 | 4.0s |
| `mug` | Кружка кофе | На столе | 0.3 | 0.2s |
| `stapler` | Степлер | На столе | 0.8 | 0.4s |
| `lamp` | Настольная лампа | На столе | 2.0 | 1.0s |

**Сила броска**: `force = chargeTime / maxCharge * (800 / mass)` px/s по X, `-600 / mass` по Y (вверх).

## Физика полёта

```
каждый кадр (dt):
  vy += gravity * dt          // гравитация 980 px/s²
  x  += vx * dt
  y  += vy * dt
  rotation += vx * 0.001 * dt // вращение пропорционально скорости

при касании пола:
  vy *= -bounce               // отскок (bounce = 0.6)
  vx *= friction              // трение (friction = 0.85)
  если |vy| < threshold → остановка

при касании стен:
  vx *= -bounce
```

Объекты остаются где упали до нажатия Reset.

## Заряд и визуальный индикатор

- При зажатии: кольцо свечения вокруг объекта заполняется (0% → 100%)
- Цвет: зелёный (слабый) → жёлтый → красный (максимум)
- При максимуме: объект вибрирует + flash-эффект
- Кот анимируется быстрее при большем заряде

## Звуки (Web Audio API — синтез, без файлов)

| Событие | Звук |
|---|---|
| Удар кота | Тупой глухой удар (oscillator burst) |
| Приземление лёгкого | Тонкий треск |
| Приземление тяжёлого | Низкий грохот |
| Бросок | Свист полёта (pitch sweep) |
| Папки | Шелест листов |
| Reset | Мягкий "whoosh" |

## UI

- **Счётчик ударов**: верхний левый угол, минималистичный
- **Кнопка Reset**: нижний правый угол, "🔄 Привести в порядок"
- **Мобайл**: touch события, tap = click, touchstart/move/end для заряда
- **Десктоп**: mousedown/up для заряда, cursor меняется на pointer над объектами
